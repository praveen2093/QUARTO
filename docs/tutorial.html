Quarto Tutorial for Beginners

Quarto Tutorial for Complete Beginners

This page teaches you Quarto step-by-step. It is built for people who are new to Quarto and may not have any background in R.

Important

You do not need R to use Quarto. Start with plain Markdown and HTML output, then add code later if you want.

1. What is Quarto?

Quarto is an open-source publishing system that turns plain text into polished documents and websites.

  • Write content in Markdown
  • Add headings, lists, links, images, and tables
  • Render to HTML, PDF, Word, slides, and more

2. Why use Quarto?

Quarto helps you:

  • create one document and output it in multiple formats
  • keep text and code together in one file
  • build websites, tutorials, and reports
  • publish clean content with version control

3. Do I need R?

No.

Quarto works with:

  • plain Markdown only
  • Python
  • R
  • Julia
  • Observable JavaScript

For beginners, start with text-only documents.

4. What do I need?

Required

  • Quarto
  • A text editor such as VS Code

Optional

  • Python or R if you want executable code blocks
  • A TeX engine like TinyTeX or TeX Live for PDF output

Check Quarto

quarto --version

If Quarto is installed, this prints a version number.

5. What is Markdown?

Markdown is a simple text format that makes writing easy.

Headings

# Main title
## Section title
### Smaller section

Bold and italic

**bold**
*italic*

Lists

- item one
- item two

Links

[Quarto website](https://quarto.org)

6. Your first Quarto document

Create a file named hello.qmd and add:

---
title: "My First Quarto Document"
format: html
---

# Hello

This is my first Quarto document.

- It is simple
- It is beginner-friendly
- I do not need to know R to start

7. What is YAML?

The section between --- lines is YAML. It sets the document title, format, and other options.

Example:

---
title: "Beginner Tutorial"
format: html
---

8. Render your document

Run this command:

quarto render hello.qmd

Quarto generates hello.html.

9. Create a website page

Quarto websites use multiple .qmd files and a _quarto.yml project file.

Example _quarto.yml:

project:
  type: website

website:
  title: "My Quarto Website"
  navbar:
    right:
      - href: index.qmd
        text: Home
      - href: tutorial.qmd
        text: Tutorial

format:
  html:
    theme: cosmo
    toc: true

Example index.qmd:

---
title: "Home"
---

# Welcome

This is my Quarto website.

11. Add code blocks

You can add code blocks that only show code:

quarto render hello.qmd

If you install Python or R, Quarto can also run code and show results.

Python example

Code 
print("Hello from Python")
Hello from Python

12. Generate multiple outputs

You can render HTML, PDF, and Word from one file.

format:
  html: default
  pdf: default
  docx: default

13. Common beginner mistakes

  • Forgetting the --- lines in YAML
  • Using the wrong file extension (.qmd instead of .txt)
  • Broken image paths
  • Trying to run code without installing Python or R

14. Beginner workflow

  1. Install Quarto
  2. Create a .qmd file
  3. Add YAML and Markdown
  4. Render to HTML
  5. View the output
  6. Edit and render again
